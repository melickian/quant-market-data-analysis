# Quantitative Market Data Analysis

This project focuses on quantitative analysis of financial market data using Python.
The goal is to explore price behavior, volatility, and relationships between assets.

## Project Structure

data/ # raw and processed datasets
notebooks/ # Jupyter notebooks for analysis
src/ # utility functions
figures/ # generated plots and charts


## Project Status
- [ ] Data collection
- [ ] Data cleaning
- [ ] Exploratory data analysis (EDA)
- [ ] Technical indicators
- [ ] Visualization

## Tools
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
